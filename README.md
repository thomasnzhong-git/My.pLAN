## English

This repository manages the holiday data used in the **My Plan** application.

### 📂 Folders and Data Guide
All 2026 holiday information for each country is stored in JSON format within the `holidays` folder.

* **[New Zealand Holidays](./holidays/2026_NZ_holidays.json)**: Official 2026 holiday data for New Zealand.
* **[Korea Holidays](./holidays/2026_ko_holidays.json)**: Official 2026 holiday data for South Korea.
* **[China Holidays](./holidays/2026_zh_holidays.json)**: Official 2026 holiday data for China.
